PlaceKeeper는 방문한 장소를 GPS로 저장하고, 사진과 메모를 첨부해 나만의 장소 기록장을 만들어주는 앱입니다.

PlaceKeeper의 핵심 기능

장소 기록을 더 쉽고 스마트하게

GPS 현재 위치 저장

버튼 하나로 현재 위치를 저장하고 카카오 API로 정확한 주소를 자동 입력해드려요.

도시 & 분류 관리

서울, 대전, 광주 등 도시별로 나누고 카페, 맛집, 직장 등 원하는 분류로 정리하세요.

사진 첨부

장소마다 사진을 여러 장 첨부해 기억을 더 생생하게 남길 수 있어요.

구글 드라이브 백업

구글 계정으로 로그인하면 내 장소 데이터를 드라이브에 자동 백업하고 언제든 복원할 수 있어요.

장소 공유

저장된 장소 주소를 카카오톡, 문자 등으로 손쉽게 공유할 수 있어요.

엑셀 / JSON 내보내기

내 장소 데이터를 엑셀이나 JSON 파일로 내보내 PC에서도 활용하세요.

개인정보처리방침

PlaceKeeper는 사용자의 개인정보를 소중히 보호합니다.

🔒 PlaceKeeper는 사용자의 위치 정보를 외부 서버에 저장하거나 전송하지 않습니다. 모든 데이터는 사용자의 기기 내부에만 저장됩니다.
